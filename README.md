# pablosuzarte.com

Personal portfolio site for Pablo Suzarte, built with Astro and deployed on Vercel.

## Current focus

- homepage positioning around UX leadership, AI, and future-ready design practice
- a dedicated internal page for `AI Project Brain Lab` at `/ai-project-brain-lab/`
- keeping the site visually coherent while expanding the portfolio into AI and education work

## Future note

`AI Project Brain Lab` is now integrated into the portfolio and live as its own page, but the content is still an early strong draft. A future pass should refine the page so the story, role definition, and teaching language become sharper and more elegant.

## Commands

| Command | Action |
| :--- | :--- |
| `npm install` | Install dependencies |
| `npm run dev` | Start local dev server at `localhost:4321` |
| `npm run build` | Build the production site into `./dist/` |
| `npm run preview` | Preview the production build locally |

## Main routes

- `/` — homepage portfolio
- `/ai-project-brain-lab/` — dedicated AI + Education project page
- `/projects/[slug]` — case study routes generated from `src/data/projects.ts`
